Description de votre projet

Par Brice Berclaz le 26/03/2020 à 20:56

Ce projet est réalisé dans le cadre du module 624-2 "Génie Logiciel - Rich Internet Application (RIA)" en filière informatique de gestion à la HES-SO Valais-Wallis. Nous avons décidé de nous orienter vers un jeu javascript sur le thème des incendies qui ont touché l'Australie en fin d'année 2019 et début d'année 2020. Le but du jeu est simple : vous incarnez un pompier qui doit aller sauver des koalas. La difficulté réside dans le fait que la forêt sur la carte du jeu s'embrasera de plus en plus au fil des secondes. Pour palier à ça et donner une chance au pompier de pouvoir mener sa mission à bien, il sera muni d'un pistolet à eau pour éteindre le feu temporairement.

Quand nous implémenterons le code nous ferons des tests pour voir la manière la plus efficace d'y jouer : par niveaux avec une difficulté ascendante ou avec un plancement aléatoire sur la carte pour chaque nouvelle partie.

Le code source de l'application est disponible sur le github du projet.
